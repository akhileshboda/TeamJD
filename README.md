# Freelance Consulting for Jake Dedert

This repository documents freelance consulting work for **Jake Dedert**, a fitness coach focused on:

- Client journey optimization
- Service positioning and business operations support

- Offer packaging for transformation and prep programs

- `design/` - Visual concepts and design-related materials
- `docs/` - Proposals, charts, and project documentation
- `requirements/` - Discovery artifacts and requirements-gathering files
- `testing/` - Validation notes and test outputs
